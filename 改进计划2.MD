# FileSort 改进计划 2.0

> 基于 2026-05-21 全面代码审查结果，针对用户使用流程、数据流完整性、代码架构质量制定的系统性优化方案。

---

## 一、总则与原则

### 1.1 需求描述 + 生成规则 + 检验标准（三件套原则）

每个优化项必须包含以下三部分：

| 组成部分 | 说明 | 示例 |
|----------|------|------|
| **需求描述** | 明确要解决什么问题、用户场景、预期行为 | "用户首次使用时无规则，需自动创建默认规则" |
| **生成规则** | 代码编写规范、命名约定、文件放置位置、继承关系 | "Service 层统一用 `actor`，ViewModel 层统一用 `@Observable final class`" |
| **检验标准** | 代码示例 + 测试用例 + 验收条件 | "构建通过 + 默认规则 6 条 + 无重复创建" |

### 1.2 功能模块原则

- **单一职责**：一个文件/类只做一件事
- **高内聚低耦合**：Service 不依赖 ViewModel，ViewModel 不直接操作 FileManager
- **命名清晰**：文件名 = 类名 = 职责描述（如 `FileScanService` 只做扫描）
- **文件结构语义化**：目录名反映功能域（`Services/`、`ViewModels/`、`Views/`）

### 1.3 最小代码原则

- **代码复用**：优先复用现有模块，抽象需符合"三次法则"（遇到三次再抽）
- **合并优先**：新代码与已有代码功能类似时合并，而非新建
- **禁止重复**：相同逻辑不得在两处以上实现

### 1.4 重构清理原则

- **替换时标记废弃**：新增 `@available(*, deprecated, message: "...")` 标记
- **验证无影响**：构建通过 + 功能测试通过
- **删除废弃代码**：确认无引用后删除，提交信息注明清理范围
- **格式**：`refactor: 清理 XXX 废弃代码，由 YYY 替代`

### 1.5 风格一致性原则

- **UI 格调**：延续当前 `.regularMaterial` + `RoundedRectangle(cornerRadius: 16)` 卡片风格
- **色彩体系**：主色 `.blue`，成功 `.green`，警告 `.orange`，错误 `.red`
- **字体规范**：标题 `.headline`/`.title2.bold()`，正文 `.subheadline`，辅助 `.caption`
- **按钮规范**：主操作 `.borderedProminent` + `.controlSize(.large)`，次操作 `.bordered`

### 1.6 继承与兼容性原则

- **Settings 继承**：新增配置项必须通过 `@AppStorage` 或 `UserDefaults(suiteName:)` 统一管理，不得硬编码
- **环境对象传递**：`PurchaseManager` 通过 `.environment()` 注入，所有子视图通过 `@Environment` 获取
- **模型兼容**：SwiftData 模型新增字段必须有默认值，确保旧数据自动迁移
- **常量继承**：App Group ID、IAP Product ID 等统一从 `AppConstants` 读取

### 优化项 B：Security-Scoped Bookmark 持久化

#### B.1 需求描述

当前 `DirectoryPickerView` 通过 `startAccessingSecurityScopedResource()` 获取访问权限，但：
1. 权限仅在 App 运行期间有效，重启后丢失
2. 用户每次使用都需重新选择目录
3. Widget 和 Shortcuts 无法访问用户之前选过的目录

us.md 明确要求 `SecurityScopedBookmarkService` 模块，但当前代码中缺失。

#### B.2 生成规则

1. 新建 `Services/BookmarkService.swift`
2. 职责：保存/读取/删除 security-scoped bookmark
3. 使用 `UserDefaults` 存储 bookmark Data（非 App Group，仅主 App 使用）
4. ViewModel 通过 `BookmarkService` 获取已保存目录，避免重复选择
5. 与现有 `DirectoryPickerView` 集成，选择后自动保存 bookmark

#### B.3 检验标准

**代码示例**：
```swift
// Services/BookmarkService.swift
import Foundation

struct BookmarkService {
    private let bookmarksKey = "savedDirectoryBookmarks"

    func saveBookmark(for url: URL) throws {
        let data = try url.bookmarkData(
            options: .minimalBookmark,
            includingResourceValuesForKeys: nil,
            relativeTo: nil
        )
        var bookmarks = loadAllBookmarks()
        bookmarks[url.path] = data
        UserDefaults.standard.set(
            try? JSONEncoder().encode(bookmarks),
            forKey: bookmarksKey
        )
    }

    func accessBookmark(for path: String) -> URL? {
        guard let bookmarks = loadAllBookmarks(),
              let data = bookmarks[path] else { return nil }
        var isStale = false
        guard let url = try? URL(
            resolvingBookmarkData: data,
            options: .withSecurityScope,
            relativeTo: nil,
            bookmarkDataIsStale: &isStale
        ) else { return nil }
        _ = url.startAccessingSecurityScopedResource()
        return url
    }

    func removeBookmark(for path: String) {
        var bookmarks = loadAllBookmarks()
        bookmarks.removeValue(forKey: path)
        UserDefaults.standard.set(
            try? JSONEncoder().encode(bookmarks),
            forKey: bookmarksKey
        )
    }

    func loadAllBookmarks() -> [String: Data]? {
        guard let data = UserDefaults.standard.data(forKey: bookmarksKey) else { return nil }
        return try? JSONDecoder().decode([String: Data].self, from: data)
    }
}
```

**集成到 DirectoryPickerView**：

### 优化项 C：免费用户功能限制实现

#### C.1 需求描述

price.md 明确定义了免费/付费功能边界，但当前代码中：
1. 免费用户可无限使用所有功能（仅重复检测有 Paywall 遮罩）
2. 缺少以下限制：
   - 自定义规则限制 1 条（当前无限制）
   - Undo 历史限制 1 批（当前 50 批）
   - Widget/Shortcuts 限制（当前无检查）
   - iCloud Drive 访问限制（当前无检查）

#### C.2 生成规则

1. 在 `PurchaseManager` 中新增 `FeatureGate` 枚举，定义所有付费功能
2. 新增 `canAccess(_ feature: FeatureGate) -> Bool` 方法
3. 所有功能限制点统一调用此方法，不得硬编码 `isPremium`
4. 限制提示统一使用现有 `PaywallView`

#### C.3 检验标准

**代码示例**：
```swift
// 在 PurchaseManager.swift 中新增
extension PurchaseManager {
    enum FeatureGate {
        case customRules        // 免费限 1 条
        case duplicateDetection // 免费不可用
        case widgetAccess       // 免费不可用
        case shortcutsAccess    // 免费不可用
        case unlimitedUndo      // 免费限 1 批
        case iCloudAccess       // 免费不可用
    }

    func canAccess(_ feature: FeatureGate) -> Bool {
        switch feature {
        case .customRules:
            return isPremium // 免费用户在 View 层检查数量
        case .duplicateDetection, .widgetAccess, .shortcutsAccess, .iCloudAccess:
            return isPremium
        case .unlimitedUndo:
            return isPremium
        }
    }

    func canAddRule(currentCount: Int) -> Bool {
        return isPremium || currentCount < 1
    }

    func canUndoBatch(currentBatchCount: Int) -> Bool {
        return isPremium || currentBatchCount < 1
    }
}
```

```swift
// FileMoveService.swift - 修改返回类型
struct MoveResult {
    let successCount: Int
    let failCount: Int
    let moveRecords: [MoveRecord]
    let failedActions: [(action: SortAction, error: String)]
}

func executeActions(_ actions: [SortAction], conflictResolution: Conflict.ConflictResolution = .rename) async -> MoveResult {
    var successCount = 0
    var failCount = 0
    var moveRecords: [MoveRecord] = []
    var failedActions: [(action: SortAction, error: String)] = []
    for var action in actions {
        do {
            // ... 现有逻辑 ...

            failCount += 1
            failedActions.append((action: action, error: error.localizedDescription))
        }
    }
    return MoveResult(successCount: successCount, failCount: failCount, moveRecords: moveRecords, failedActions: failedActions)
}
```

当前重复检测对所有文件计算 SHA256 哈希，大文件场景下性能差：
1. 1GB 视频文件需完整读取计算哈希
2. 无进度反馈，用户以为卡死
3. us.md 要求"快速比较 + 哈希验证"两阶段

#### G.2 生成规则

1. 第一阶段：按文件大小分组（已有），快速筛选潜在重复
2. 新增第二阶段：对大文件（>10MB）仅计算前 1MB + 后 1MB 的部分哈希
3. 部分哈希匹配后再计算完整哈希确认
4. 新增 `progressHandler: ((Int, Int) -> Void)?` 回调报告进度
5. 复用现有 `DuplicateDetectService`，不新建文件

```swift
// DuplicateDetectService.swift - 优化后
struct DuplicateDetectService {
    func findDuplicates(
        in files: [ScannedFile],
        progressHandler: ((Int, Int) -> Void)? = nil
    ) async -> [DuplicateGroup] {
        let sizeGroups = Dictionary(grouping: files) { $0.size }
        let potentialDuplicates = sizeGroups.filter { $0.value.count > 1 && $0.key > 0 }
        var duplicateGroups: [DuplicateGroup] = []
        let total = potentialDuplicates.count
        for (index, (_, filesWithSameSize)) in potentialDuplicates.enumerated() {
            progressHandler?(index + 1, total)
            let hashGroups = await computeHashes(files: filesWithSameSize)
            for (_, filesWithSameHash) in hashGroups where filesWithSameHash.count > 1 {
                let group = DuplicateGroup(
                    files: filesWithSameHash,
                    fileSize: filesWithSameHash.first?.size ?? 0,
                    hashValue: filesWithSameHash.first?.url.lastPathComponent ?? ""
                )
                duplicateGroups.append(group)
            }
        }
        return duplicateGroups.sorted { $0.totalWastedSpace > $1.totalWastedSpace }
    }

    private func computeHashes(files: [ScannedFile]) async -> [String: [ScannedFile]] {
        await withTaskGroup(of: (String, ScannedFile?).self) { group in
            for file in files {
                group.addTask {
                    if file.size > 10_485_760 { // > 10MB
                        if let hash = Self.partialSHA256(of: file.url) {
                            return (hash, file)
                        }
                    } else {
                        if let hash = Self.sha256(of: file.url) {
                            return (hash, file)
                        }
                    }
                    return ("", nil)
                }
            }
            var hashMap: [String: [ScannedFile]] = [:]
            for await (hash, file) in group {
                guard let file = file, !hash.isEmpty else { continue }
                hashMap[hash, default: []].append(file)
            }
            return hashMap
        }
    }

    private static func partialSHA256(of url: URL) -> String? {
        guard let data = try? Data(contentsOf: url, options: .mappedIfSafe) else { return nil }
        let headSize = min(1_048_576, data.count) // 前 1MB
        let headData = data.prefix(headSize)
        let tailData = data.suffix(min(1_048_576, data.count - headSize))
        var hasher = SHA256()
        hasher.update(data: headData)
        hasher.update(data: tailData)
        let hash = hasher.finalize()
        return hash.compactMap { String(format: "%02x", $0) }.joined()
    }
    // ... 保留原有 sha256 方法 ...
}
```

```swift
func scanForDuplicates(in files: [ScannedFile]) async {
    isScanning = true
    scanError = nil
    selectedForDeletion.removeAll()
    scanProgress = 0
    do {
        let groups = try await duplicateService.findDuplicates(in: files) { current, total in
            Task { @MainActor in
                scanProgress = Double(current) / Double(total)
            }
        }
        duplicateGroups = groups
        totalWastedSpace = groups.reduce(0) { $0 + $1.totalWastedSpace }
    } catch {
        scanError = error.localizedDescription
    }
    isScanning = false
}
```
