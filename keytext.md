## Promotional Text
Tame your messy folders in one tap. Smart rules, duplicate detection, and full undo -- your files, perfectly organized.

## Description
Is your Downloads folder a disaster? iCloud Drive cluttered with random files? FileSort is the automatic file organizer your iPhone has been waiting for.

Unlike traditional file managers that only let you browse, FileSort actively sorts your files for you. Open the app, see the mess, tap once, and watch everything fall into place.

KEY FEATURES

- ONE-TAP SMART SORT
Scan any folder and let FileSort automatically organize files by type -- documents, images, videos, archives, audio, and code. No setup required. Just tap Sort Now.

- CUSTOM RULES ENGINE
Create powerful rules without writing a single line of code. Set conditions like "if file type is PDF and size is over 5MB, move to Work/Contracts." Drag to reorder priority. First match wins.

- DUPLICATE FILE DETECTION
Reclaim wasted storage by finding and removing duplicate files. FileSort uses SHA-256 hashing for accurate detection, so you never accidentally delete the wrong file.

- FULL UNDO HISTORY
Changed your mind? No problem. Undo any sort batch with a single tap. Every move is reversible, so you can sort with confidence.

- SORT PREVIEW
See exactly what will happen before it happens. Review every file move, spot conflicts, and adjust before executing. No surprises.

- HOME SCREEN WIDGET
Add the FileSort widget to see your messy file count at a glance and trigger a sort right from your home screen.

- SIRI SHORTCUTS
Automate your file organization with Shortcuts. Create workflows that sort files on a schedule or trigger sorting with your voice.

- iCLOUD DRIVE INTEGRATION
Deep integration with iCloud Drive means your files stay in sync across all your Apple devices. Sort once, see it everywhere.

- CELEBRATION & STATS
Watch a satisfying celebration animation when your sort completes. See detailed stats on how many files were organized and how much space was reclaimed.

FREE VS PREMIUM

Free tier includes unlimited scanning, one-tap basic sort, sort preview, and one undo batch. Upgrade to Premium for unlimited custom rules, duplicate detection, home screen widget, Siri Shortcuts, unlimited undo history, and iCloud Drive deep integration.
